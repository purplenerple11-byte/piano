If it isn't working

  1. Are you in the right browser? On iPad, Safari can never do this. Use the free Web MIDI Browser and load this page in it. (On a computer, use Chrome or Edge.)
  2. By cable: use a genuine Apple USB Camera Adapter plus the piano's USB-B cable. Nothing to pair — the piano should just appear. If it doesn't, the adapter is the usual culprit (clones often don't pass enough power).
  3. By Bluetooth: pair inside the app, using its own Bluetooth MIDI button. This is the step nearly everyone misses — Bluetooth MIDI devices never appear in iOS Settings → Bluetooth. Confirmed in the FP-10 manual. The FP-10's Bluetooth is on by default; [FUNCTION] lights when a connection is live.
  4. Order matters (Bluetooth): piano on → open the app → pair → then open this page.
